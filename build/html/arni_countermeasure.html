class arni_countermeasure.constraint.Constraint(name, constraint_root, planned_reaction, min_reaction_interval, reaction_timeout)¶

Contains the whole constraint with corresponding reactions.

evaluate_constraint(storage)¶

Evaluates this constraint and sets the attributes according to the result of the evaluation.

Parameters:storage (RatedStatisticStorage) – The storage where the incoming statistics are saved.
evaluation_result = None¶

True if the current state of the constraint says that an execution of the reactions is necessary.

notify_of_execution()¶

Tells this constraint that its reactions have just been executed.

# Note: its important to notify this constraint so it knows how long the last execution was ago and does not execute too often.

planned_reaction = None¶

An list of reactions that should be executed if the constraint has been true longer than min_reaction_interval milliseconds. :type: list of Reactions

true_since = None¶

Time since when this constraint is true. # Note if true_since is 0 it has not been true. :type: rospy.Time

get_outcome(seuid, statistic_type)¶

Return the outcome of the specific seuid and statistic_type.

Parameters:
  • seuid (string) – The seuid of the entity the statistic belongs to.
  • statistic_type (string) – Type of statistic. Arrays get a suffix: type_0 type_1 and so on.
Returns:

Return type:

Outcome (int)
