Definitions¶

Seuid¶

A seuid (statistics entity unique identifier) identifies a certain node, host, connection (a topic, a subscriber and a publisher) or a topic following the this pattern:

  • Hosts: h!127.0.0.1
  • Nodes: n!/my_node
  • Topics: t!/topic_name
  • Connections: c!subscriber!topic!publisher

get_seuid(field)¶

Returns the seuid for a specified component.

Parameters:

field – Can be “host”, “topic”, “subscriber”, “publisher” or “node”. Only if the SEUID object contains this key
Returns:

A seuid (str)
Raises:
  • KeyError – When the current SEUID does not contain the given field since it has a different type
  • AttributeError – When the given parameter is none of the above.
